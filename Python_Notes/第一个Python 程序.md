## 第一个 helloPython 程序

### Python 源程序的基本概念

- 一个特殊格式的文本文件
- 结尾通常是`.py`

#### 任务步骤

- 在桌面下，新建`认识Python`目录
创建文件夹
	mkdir 认识Python

- 在`认识Python`目录下新建 `01-hellopython.py`

切换到目录
	cd 认识Python

新建文件
	touch 01-hellopython.py

- 使用`gedit` 编辑 `01-hellopython.py` 并且输入一下内容：

	print (“ hello python”)
	print（“ hello  world ” ）

执行
	python 01-hellopython.py


print函数
